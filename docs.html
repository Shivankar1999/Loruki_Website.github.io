Install CLI

Requirements

  • Window 10 , Mac , Osx , Linux
  • Node.js V12 or higher
  • Vue or react front end tools

Instll

Mac (Homebrew)

$ brew install Loruku-CLI

Npm

$ Npm install Loruku-CLI

Yarn

$ Yarn install Loruku-CLI
